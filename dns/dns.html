DNS - Domain Name System

Porque usar DNS?

Quando hospedamos um site, aplicação, conta de email ou qualquer coisa na internet na internet temos que atribuir DNSs ao dominio do site, são od DNSs que vão determinar onde um site está hospedado

DNS é uma infra estutura hierárquica e para encontrar o ip do dominio ele faz buscas resursisvas e uma curiosidade é que cada dominio possui um ponto (.) no final ou seja

Como funciona DNS

Consulta: Quando inserimos um nome de dominio como mikemarciano.dev.br no navegador, o computaodr envia uma consulta DNS para um resolver de DNS que pode ser um DNS local ou provedor publico como o Google

Resolvendo a Consulta resolvedor DNS passa a fazer consultas con uma sequencia de servidores DNS, começando pelos servidores raiz, em busca do endereço IP associado ao nome de dominio determinado.

Verificação de Cache resolvedor primeiro verifica seu cache local para verificar se já possui o endereço IP do dominio solicitado
