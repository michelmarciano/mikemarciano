Propriedades para Texto em CSS

Propriedade Color

A propriedade CSS color define a cor do texto de um elemento HTML. Esta propriedade pode receber valores em diferentes formatos como: nomes predefinidos (red, blue), hexadecimal (#FF0000), RGB (rgb(255, 0, 0)), RGBA para transparência (rgba(255, 0, 0, 0.5)), HSL (hsl(0, 100%, 50%)) e HSLA. Cada formato oferece diferentes maneiras de especificar cores, permitindo maior precisão e controle na estilização. referencias: mikemarciano.dev.br

Propriedade Font-Size

A propriedade CSS font-size controla o tamanho da fonte de um elemento HTML. Pode ser definida usando unidades absolutas como pixels (px) ou pontos (pt), ou unidades relativas como em (relativo ao elemento pai), rem (relativo ao elemento root), porcentagem (%) e viewport units (vw, vh). As unidades relativas são recomendadas para melhor responsividade e acessibilidade do site.

See the Pen Propriedade de Texto - FontSize by Michel Oliveira Marciano (@Michel-Oliveira-Marciano) on CodePen.

Propriedade Font-Weight

A propriedade CSS font-weight controla a espessura (peso) da fonte de um elemento HTML. Pode ser definida usando valores numéricos de 100 a 900 (onde 400 é normal e 700 é bold), ou palavras-chave como normal, bold, bolder e lighter. Esta propriedade é fundamental para criar hierarquia visual e destaque em textos.
