Propriedades para Texto em CSS

Propriedade Color

A propriedade CSS color define a cor do texto de um elemento HTML. Esta propriedade pode receber valores em diferentes formatos como nome da cor, hexadecimal, RGB ou HSL.

Propriedade Font-Size

A propriedade CSS font-size define o tamanho da fonte de um elemento HTML. Pode ser definida usando diferentes unidades como pixels (px), em, rem, porcentagem, entre outras.
