Spring IoC Container

Spring IoC é responsavel por instanciar, configurar e injetar as dependencias de objetos em outros objetos, quando executamos uma aplicação spring o IoC container é inicializado e começa a instanciar os beans que ele precisa gerenciar, Bean é o nome dado para objetos gerenciados pelo container do Spring, o container de inversão de controle é representado pela interface chamada aplication context.
