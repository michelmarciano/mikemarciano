Conectar Gmail

    Criar Nova API

  • Acesse console do google cloud

  • Clique em NOVO PROJETO

  • Selecione o projeto que acabou de criar e clique em Apis e Serviços

  • Faça uma busca por Gmail e selecione Gmail Api

    • Credenciais

    Clique em criar credencial

  • Selecione Chave de API
  • De um nome para sua chave OAuth 2.0

    • Escopos

  • Clique em Adicionar ou remover Escopos
  • Faça uma busca por Gmail
  • Clique em Salvar e continuar

    • ID do cliente OAuth

  • Selecione o tipo de aplicativo como Aplicativo Web
  • Coloque o nome do cliente
  • Coloque a UrI de redirecionamento autorizado "URL do N8N"
  • Concluir

    • Publicar Aplicativo

  • Agora basta ir na tela de Publico Alvo e clicar em Publicar aplicativo
